# mysql-bq-load-test

Simple script to load a MySQL table into BQ. Extracts schema and data as JSON.

## Usage

```
mysql-bq-load-test
Usage: mysql-bq-load-test [options]

  -d, --database-url <value>
                           the URL, e.g. 'jdbc:mysql://localhost:3306/test'
  -u, --username <value>   
  -p, --password <value>   
  -t, --table <value>      
  -o, --out <value>   
```
